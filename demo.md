### 電腦版

`{{xxx}}` 是動態的可以後台填寫

```html
<iframe
  src="https://mg4electric.netlify.app/m/"
  height="{{xxx}}" // 這裏
  width="100%"
  style="border: none; width: 100%; height: {{xxx}}px" // 這裏
  frameborder="0"
  scrolling="no"
  seamless="seamless"
></iframe>
```
